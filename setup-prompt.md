# Setup Prompt

Use the version that matches the coding agent you are using. Claude Code and Codex do not use the same setup commands or project instruction files, so copying the wrong prompt can cause confusing behavior.

If you launched the agent with `claude`, use the Claude Code version. If you launched it with `codex`, use the Codex version.

Copy one prompt only:

## Claude Code

Use this if you are working in Claude Code. It tells Claude to run `/init` and update `CLAUDE.md`.

[Open the Claude Code Setup Prompt](setup-prompt-claude.md)

## Codex

Use this if you are working in Codex. It tells Codex to create or update `AGENTS.md` directly and avoids Claude-specific slash commands.

[Open the Codex Setup Prompt](setup-prompt-codex.md)
